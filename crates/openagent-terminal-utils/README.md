# openagent-terminal-utils

Utility functions for OpenAgent Terminal (themes, snippets, migration).

Features:
- themes (default)
- snippets (default)
- migrate
- all

Testing:
- cargo test -p openagent-terminal-utils
- With optional features:
  - cargo test -p openagent-terminal-utils --features migrate
