# plugin-system

Unified plugin system for OpenAgent Terminal with WASM runtime and host integration.

Features:
- wasm-runtime (WASI runtime + networking for host_net_fetch)
- host-integration (native plugin loading via libloading)

Testing:
- cargo test -p plugin-system
