# openagent-terminal-core

Alacritty-derived terminal core used by OpenAgent Terminal.

Features:
- serde: enable Serde support for config and state (default)

Testing:
- Run unit tests for this crate:
  - cargo test -p openagent-terminal-core

- See the workspace testing guide for linting and GUI/headless test notes:
  - ../docs/TESTING.md
  - ../docs/features.md
